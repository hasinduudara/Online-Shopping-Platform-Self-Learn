# Online Shopping Platform 🛒

A modern, full-stack e-commerce web application built with a **Next.js** frontend and a **Spring Boot** backend. The entire application is containerized using **Docker** and deployed on a **Microsoft Azure Virtual Machine**.

## 🚀 Tech Stack

* **Frontend:** Next.js, React, Node.js (v20)
* **Backend:** Spring Boot (Java 21), Maven
* **Database:** PostgreSQL (v15)
* **Deployment & DevOps:** Docker, Docker Compose, Microsoft Azure (Linux VM)
